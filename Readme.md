# Project - Library
    This project sets to emulate a functioning library with dB's for the books, clients and loans.

## type explaination
    The book type set the maximum loan time for the book: 
    • 1 – up to 10 days 
    • 2 – up to 5 days 
    • 3 – up to 2 days
